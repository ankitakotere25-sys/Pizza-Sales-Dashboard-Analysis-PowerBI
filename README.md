#  Power BI Dashboard Project

##  Project Title
**Pizza Sales Dashboard Analysis**



##  Project Description
This project is an interactive Power BI dashboard designed to analyze pizza sales data. It provides valuable insights into sales performance, customer preferences, and business trends using visual analytics.

The dashboard helps stakeholders make data-driven decisions by presenting key metrics and patterns in an intuitive format.



##  Objectives
- Analyze overall sales performance
- Identify top-selling pizzas and categories
- Track revenue trends over time
- Understand customer ordering behavior
- Provide actionable business insights



##  Dataset Information
The dataset includes:
- Order ID
- Pizza Name
- Category (Veg/Non-Veg)
- Size (Small, Medium, Large)
- Quantity Sold
- Price
- Order Date & Time



##  Key Metrics (KPIs)
- 💰 Total Revenue
- 🍕 Total Pizzas Sold
- 📦 Total Orders
- 📊 Average Order Value
- ⏱ Orders by Time (Hourly Trends)



##  Dashboard Features
- Sales trend analysis (daily/monthly)
- Top & bottom performing pizzas
- Category-wise sales distribution
- Size-wise performance analysis
- Time-based order insights (peak hours)
- Interactive filters and slicers



##  Tools & Technologies Used
- **Power BI Desktop**
- Data Cleaning & Transformation using Power Query
- DAX (Data Analysis Expressions)
- Data Visualization Techniques
